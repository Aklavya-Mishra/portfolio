# Aklavya Mishra - Terminal Portfolio

A hacker-style terminal portfolio. Because regular portfolios are too mainstream. Boot it up, type commands, pretend you're in The Matrix. 🖥️

## Features

- 3-second loading screen (for dramatic effect)
- Fake boot sequence with ASCII art
- Fully interactive CLI interface
- Command history (up/down arrows)
- Tab autocomplete
- Quick command buttons for the mouse lovers
- Responsive design
- GitHub dark theme aesthetics

## Available Commands

| Command | Description |
|---------|-------------|
| `help` | Show all available commands |
| `about` | Learn about me |
| `skills` | View technical skills |
| `experience` | View work experience |
| `projects` | See my projects |
| `contact` | Get contact information |
| `neofetch` | Display system info (try it!) |
| `clear` | Clear the terminal |
| `whoami` | Who am I? |
| `date` | Show current date |
| `echo [text]` | Echo back text |
| `ls` | List files |
| `cat [file]` | Read a file (try `cat about.txt`) |
| `pwd` | Print working directory |
| `sudo` | Nice try 😏 |
| `exit` | You can't escape |

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- **React** 19.2.4
- **Vite** 8.0.1
- **JetBrains Mono** font
- Pure CSS (no UI libraries)
